## 🛠️ **QUICK COMMANDS**

### **Server Control**
- **Stop Server**: Press `Ctrl + C` in terminal
- **Restart Server**: `npm run dev`
- **Change Port**: `npm run dev -- -p 3001`

### **Build for Production**
```bash
npm run build    # Build optimized version
npm start        # Run production server
```
